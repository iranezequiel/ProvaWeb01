O que é o protocolo HTTP?

O HTTP (HyperText Transfer Protocol) é o protocolo responsável pela comunicação entre o navegador e os servidores da internet. Sempre que você acessa um site, o navegador envia uma requisição HTTP para o servidor e o servidor responde enviando a página solicitada.

Como o HTTP funciona na Web

Quando digitamos um endereço como www.google.com, o navegador envia um pedido para o servidor. Esse pedido é chamado de request. O servidor recebe esse pedido e envia uma resposta chamada response, que normalmente é uma página HTML.

Imagem mostrando a comunicação entre cliente e servidor usando HTTP.

Exemplo de uma requisição HTTP

Exemplo de request enviado pelo navegador:

GET /index.html HTTP/1.1
Host: www.site.com
User-Agent: Chrome
Accept: text/html

Explicação dos campos

  • GET → tipo de requisição
  • /index.html → página solicitada
  • HTTP/1.1 → versão do protocolo
  • Host → endereço do site
  • User-Agent → navegador utilizado
  • Accept → tipo de conteúdo que o navegador aceita

Exemplo de resposta do servidor (Response)

HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1200


    
        Página inicial
    
    
        Bem-vindo ao site!

Explicação dos campos do response

  • HTTP/1.1 200 OK → resposta do servidor (200 = deu certo)
  • Content-Type → tipo do conteúdo enviado
  • Content-Length → tamanho da resposta
  • HTML → página enviada para o navegador
